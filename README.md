# GRAB (Grade Rating Automatic Bot)

🎬 **AI 기반 영상물 등급 분류 시스템**

영상물 등급 위원회의 분류 체계를 기반으로, GPT 및 CLIP 등 최신 AI 모델을 활용하여 영상물의 내용을 자동으로 분석하고 연령 등급을 예측하는 서비스입니다.

---

## 🔍 프로젝트 개요

- **기존 문제점**
  - 수작업 중심의 등급 분류 시스템 → 시간, 비용, 인적 자원 부담
  - 신생 OTT에서 자체등급분류 체계 미확립
  - 주관적 판단에 기반한 등급 분류

- **목표**
  - 주제, 대사, 폭력성, 선정성, 공포, 약물, 모방 위험 등 7가지 요소에 대해 **자동 등급 분류 시스템 구축**
  - 웹 기반의 직관적 사용자 인터페이스 제공
  - 영상물에 대해 **객관적이고 일관된 등급 판정** 지원

---

## 🧠 사용된 모델

| 요소 | 사용 모델 | 입력 유형 |
|------|------------|-----------|
| 주제 | GPT-4o | Text |
| 대사 | KcELECTRA | Text |
| 약물(음주/흡연/마약) | CLIP / OWLv2 | Image |
| 공포 | CLIP | Image |
| 폭력성 | GPT-4o + CLIP | Text + Image |
| 선정성 | GPT-4o + CLIP | Text + Image |
| 모방 위험 | GPT-4o | Text + Image |

---

## 🧩 시스템 구조

1. **입력**: 영상 파일 (.mp4), 메타데이터
2. **전처리**
   - FFmpeg로 이미지 및 오디오 추출
   - Whisper를 통한 음성 → 대사 텍스트 변환
3. **모델별 분류**
   - 각 요소에 대해 Text/Image 기반 모델 적용
   - Prompt 기반 등급 분류 수행
4. **최종 판정**
   - 7개 요소 중 가장 높은 등급을 영상 최종 등급으로 판정
5. **시각화 및 리포트 출력**
